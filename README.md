## Account Profiler

* ruby
* Make
* rspec

# Running:
* bin/account_profiler sample-tim.txt 000-0000-0000

# Testing:
* make test

# Install:
* make
* sudo make install
* then can execute: account_profiler sample-tim.txt 000-0000-0000
